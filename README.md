# 你的名字的个人网站

使用 GitHub Pages + Jekyll (minima 主题) 构建的个人静态网站。

## 本地预览

```bash
bundle install
bundle exec jekyll serve
```

然后访问 http://localhost:4000

## 目录结构

- `_config.yml` - 站点配置
- `about.md` - 个人简介
- `index.md` - 首页（博客列表）
- `_posts/` - 博客文章（命名格式：YYYY-MM-DD-title.md）

## 部署

直接 push 到 GitHub 仓库，在仓库 Settings → Pages 选择 main 分支 /root 即可。
