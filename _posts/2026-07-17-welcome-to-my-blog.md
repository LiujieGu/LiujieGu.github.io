欢迎来到我的博客！这是用 GitHub Pages + Jekyll 搭建的第一篇文章。

你可以在 `_posts/` 目录下按 `YYYY-MM-DD-title.md` 的命名格式创建新的文章，
每篇文章开头加上上面的 front matter（layout / title / date / categories）即可。

```bash
bundle exec jekyll serve
```

运行上面命令可以在本地预览网站：http://localhost:4000
